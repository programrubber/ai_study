### BE
#### uv?
 - uv는 파이썬의 여러 도구를 하나로 통합함
 - pip (패키지 설치), pip-tools(의존성 관리), virtualenv(가상환경), pyenv(Python 관리)
 - uv venv 가상화 환경에 포팅함
#### uvicorn?
 - Python ASGI 웹 서버 (비동기 웹 애플리케이션 실행용)
 - --reload: 코드 변경시 자동으로 서버 재시작 (개발모드)
#### BE 설정해보기
``` python
uv venv --python 3.12 // 가상환경을 생성하고 python 3.12로 지정
.\.venv\Scripts\activate // 윈도우에서 가상환경을 활성화 하는 명령어
uv pip install -r requirements.txt // 파일에 필요 라이브러리가 정의 되어 있음
uvicorn app.main:app --reload
```

### 프롬프트 엔지니어링
---
* 구조적(마크다운)으로 시작한다. 
	* 역할, 목표, 과제, 유의사항, 출력 포맷, 참고 자료, 예시
* AI에게 한번에 시킬 과제
	* 싱클턴: 
	* 멀티턴: TASK를 나눠서 진행, TASK 진행 중간에 사람이 중간에 개입해야 할 때.
* AI를 어시스턴트로 활용하기
	* 결정-> 다양한 시선으로 분석(현실, 리스크 관리, 성과)
	* 아이디어 N개
* Role-playing
* Few-shot: 예시를 2개 이상
	* zero-shot: '천천히 생각하라', '단계적으로 세분화해라'
* CoT(생각의 사슬): 작업의 순서 명시
	* plan
	* dev
	* unit
	* commit
* 정형차원 출력
	* json 형태가 효율적
* meta prompting
	* 프롬프트 피드백, 수정, 생성
	* 장황하게
		* 필요한, 부수적
		* 내가원한 형식 (md, 불릿포인트, 뎁스는 2뎁스로...)
* Prompt Function
	* 프롬프트를 함수처럼 재활용한다.
	* [web] <- 웹 검색해서 
	* [detail] <- 상세 예시
	* [bref] <- 2줄 요약
