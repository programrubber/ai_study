🏗️ "돈의 흐름" 서비스 설계
핵심 가치

경제 뉴스를 분석하여 자금 이동 방향을 시각적으로 예측하는 거시경제 대시보드


1. 정보 구조 (IA)

돈의 흐름
├── 🏠 메인 대시보드
│   ├── 자금 흐름 다이어그램 (핵심 시각화)

│   └── 오늘의 흐름 한줄 요약
│
├── 📰 뉴스 & 분석
│   ├── 최신 경제 뉴스 피드
│   ├── AI 뉴스 분석 결과

│
├── 📊 자산군별 상세

2. 핵심 기능 상세
① 뉴스 수집 → AI 분석 파이프라인
뉴스 수집(웹 검색) → 카테고리 분류(금리/물가/고용/무역 등) → AI 분석(영향도 판단) → 자산군별 시그널 생성(-2 ~ +2) → 시각화

② 자금 흐름 시그널 체계
시그널의미시각 표현+2강한 유입굵은 초록 화살표 ▲▲+1유입초록 화살표 ▲0중립회색 점선 ―-1유출빨간 화살표 ▼-2강한 유출굵은 빨간 화살표 ▼▼

③ 뉴스-자금흐름 연결 로직
각 뉴스가 어떤 자산군에 어떤 방향으로 영향을 주는지 AI가 분석하고, 그 **근거(reason)**를 함께 제시. 사용자가 "왜 이 방향인지" 항상 추적 가능.

3. 데이터 소스 설계
구분소스방식경제 뉴스웹 검색 API (Claude web search)실시간거시지표한국은행 ECOS, FREDAPI 연동 또는 정적 데이터AI 분석Claude API뉴스 입력 → 시그널 JSON 출력과거 데이터persistent storage예측 기록 저장

4. AI 분석 프롬프트 설계 (핵심)
AI에게 뉴스를 넘길 때 요청할 출력 형식:
json{
  "summary": "전체 시장 상황 요약",
  "flows": [
    {
      "asset": "주식",
      "signal": 2,
      "reason": "금리 인하로 성장주 밸류에이션 매력 증가",
      "related_news": [0, 3],
      "confidence": 0.8
    }
  ],
  "insights": ["투자 시사점 1", "주의사항 2"],
  "risk_factors": ["반대 시나리오 리스크"]
}
confidence 값을 포함해서 AI도 확신하지 못하는 영역을 사용자에게 투명하게 보여줌.

5. UI/UX 핵심 포인트

중앙에 자금 흐름 다이어그램 — 6개 자산군이 원형으로 배치, 중앙에서 화살표로 연결
다크 모드 기본 — 금융 대시보드 느낌
뉴스 카드 클릭 시 해당 자산 하이라이트 — 뉴스와 자금흐름의 인과관계를 직관적으로
면책 고지 — "AI 분석이며 투자 조언이 아닙니다" 명시
